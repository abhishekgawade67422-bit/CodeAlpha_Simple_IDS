# Simple Intrusion Detection System (IDS)
This project is a Python-based tool that monitors network traffic to detect suspicious patterns.

## Features
- Detects SYN scans (Port discovery attempts).
- Monitors ICMP traffic for potential floods.
- Real-time alerting with Source and Destination IP tracking.

## Technical Details
- Language: Python
- Libraries: Scapy
